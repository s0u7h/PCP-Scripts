# REACLIPS

## Build and browse a personal sound/clip/loop library in REAPER

These scripts provide functionality for REAPER similar to Ableton Live Clips or Studio One musicloops. Musical ideas and clips can be quickly saved, then browsed and organised in the Media Explorer, and loaded into an open project as exploded tracks or as subprojects.
